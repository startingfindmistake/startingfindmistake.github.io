## C language
```c
#include <stdio.h>
```
이 줄은 본격적인 컴파일이 시가되기 전에 전처리기(Preprocessor)에게 내리는 명령입니다.  
* `#`(해시기호): C언어에서 `#`으로 시작하는 모든 줄은 '전처리기 지시자'입니다.  
컴파일러가 코드를 기계어로 번역하기 전에, 소스 코드를 편집(준비)하라는 뜻입니다.

</br>

* `include`(포함하다): "특정 파일의 내용을 이 자리에 그대로 복사해서 붙여넣어라"라는 명령어 입니다.

</br>

* `< >`(꺾쇠 괄호): 포함할 파일을 찾을 위치를 지정합니다.  
꺽쇠 괄호를 사용하면 컴파일러가 기본적으로 제공하는 **표준 라이브러리 디렉토리**에서 파일을 찾습니다.
(참고로 `" "`큰 따옴표를 사용하면 개발자가 현재 작업 중인 디렉토리에서 먼저 찾습니다.)  


<br/>

* `stdio.h`(Standard Input / Output header): '표준 입출력 헤더 파일' 입니다.
우리가 흔히 화면에 글자를 출력하는 `printf()`나, 키보드로 입력을 받는 `scanf()` 같은 함수들이 어떻게 생겼는지(선언)가 이 파일 안에 정의되어 있습니다.

요약: "컴파일을 시작하기 전에, 화면에 글자를 출력하거나 입력받는 데 필요한 기본 도구를(`stdio.h`)를 코드 맨 위에 복사해둡니다."

</>
</br>

```c
int main(int argc, char *argv[])
```
#### 메인 함수 정의
이 줄은 운영체제(Windows, Linux 등) 프로그램을 실행할 때 가장 먼저 호출하는 진입점(Entry Point)입니다.
* `int`(Integer, 정수)  
이 함수의 반환형(Return Type)입니다.
`main`함수가 모든 작업을 끝내고 운영체제에게 결과값으로 '정수'를 돌려주겠다는 뜻입니다.
보통 프로그램이 문제없이 정상적으로 종료되었을 때는 `return 0;`을 통해 `0`을 반환하고, 에러가 발생했을 때는 `0`이 아닌 다른 숫자를 반호나하여 운영체제에 상태를 알립니다.

</br>

* `main`(메인함수)  
C언어 프로그램의 시작점으 의미하는 특별히 예약된 함수 이름입니다.  
C프로그램은 반드시 하나의 `main` 함수를 가져야 하며,
프로그램이 실행되면 운영체제는 무조건 이 `main`함수부터 찾아서 실행합니다.

</br>

* `( ...)` (매개변수, 파라미터)  
프로그램을 실행할 때 외부(예: 터미널이나 명령 프롬프트)에서 프로그램 내부로 전달해 주는 입력값들입니다. 이를 명령행 인자(Command-line arguments)라고 부릅니다.

  * `int argc`(Argument Count)  
  프로그램을 실행할 때 전달된 인자의 총 개수를 나타내는 정수입니다.  
  아무런 인자를 주지 않고 프로그램만 실행해도 프로그램의 실행 경로(이름)자체가 첫 번째 인자로 돌아가기 때문에, `argc`는 항상 1이상의 값을 가집니다.
  </br>
  * `char *argv []`(Argument Vector). 
  전달된 인자들의 실제 내용(문자열)을 담고 있는 배열입니다.  
    * `char *`는 C언어에서 문장열(문자들의 배열을 가리키는 포인터)을 의미합니다.
    * `[]`는 이러한 문자열들이 여러 개 들어있는 배열이라는 뜻입니다.
    * 예를 들어 터미널에서 `./my_program hello world`라고 실행했다면
        * `argc`는 `3`dl ehlqslek.
        * `argv[0]`은 `"./my_program"`(프로그램 이름)
        * `argv[1]`은 `"hello"`
        * `argv[2]`는 `"world"`가 됩니다.

</br>
</br>
"컴파일 전에 표준 입출력 기능(`stdio.h`)을 쓸 수 있게 준비해 주고(`#include`), 운영체제가 이 프로그램을 실행할 때는 여기서부터 시작(`main`)하는데, 실행할 때 외부에서 전달된 값의 개수 (`argc`)와 그 내용물 (`argv`)을 받아서 처리할 것이며, 프로그램의 끝날 때는 운영체제에게 성공/실패 여부를 정수(`int`)로 알려주겠다."



## 논리연산 문제
---

```c
#include <stdio.h>
int main(int argc, char *argv[]) {
    int a = 5, b = 3, c = 12;
    int t1, t2, t3;
    t1 = a && b;
    t2 = a || b;
    t3 = !c;
    printf("%d", t1 + t2 + t3);
    return 0;
}
```

C언어 특징
1. 참(True)과 거짓(False)의 판단 기준
  * `0` = 거짓(False)입니다.
  * `0이 아닌 모든 숫자`(음수 포함) = 참(True)으로 취급합니다.  
  (ex: 5, 3, 12, -1)

</br>

2. 논리 연산의 결과값
  * 논리 연산(&&, ||, !)을 수행한 결과가 참이면 무조건 `1`을 반환합니다.
  * 논리 연산 결과가 거짓이면 무조건 `0`을 반환합니다.


Step1. 변수 초기화

```c
int a = 5, b = 3, c = 12;
```
* 세개의 정수형 변수 a, b, c,에 각각 5, 3, 12가 저장됩니다.

Step2: `t1`계산 (논리 AND 연산)

```c
t1 = a && b;
```
* `&&`는 AND(그리고)연산자 입니다. 양쪽이 **모두 참일 때만**결과가 참이 됩니다.
* `a`는 `5`(참)이고, `b`는 `3`(참)입니다.
* `참 &&참`이므로 결과는 **참(true)**입니다.
* 참의 결과값으로 `1`이 반환되어 `t1`에 저장됩니다. (결과 t1 = 1)

</br>

Step3 : t2계산 (논리 OR연산)

```c
t2 = a || b;
```
* `||`는 OR(또는)연산자 입니다. 양쪽 중 **하나라도 참이면** 결과가 참이 됩니다.
* `a`는 `5`(참)이고, `b`는 `3`(참)입니다.
* `참 || 참` 이므로 결과는 **참**입니다.
* 참의 결과값으로 `1`이 반환되어 `t2`에 저장됩니다.(결과: `t2 = 1`)

</br>

Step4: `t3` 계산 (논리NOT 연산)

```c
t3 = !c;
```
* `!`는 **NOT(부정)**연산자 입니다. 참을 거짓으로, 거짓을 참으로 반전시킵니다.
* `c`는 `12`이므로 **참**입니다.
* `!참` 이므로 결과는 **거짓**으로 뒤집힙니다.
* 거짓의 결과값으로 `0`이 반환되어 `t3`에 저장됩니다. (결과:`t3 = 0`)

</br>

Stpe5: 최종 계산 및 출력
```C
printf("%d", t1 + t2 + t3);
```

* 앞서 구한 값들을 모두 더합니다. `t1(1) + t2(1) + t3(0)`
* `1 + 1 + 0 = 2`
* `%d` (10진수 정수형 포맷)를 통해 최종적으로 숫자 `2`가 화면에 출력됩니다.
