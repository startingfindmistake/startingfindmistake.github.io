```python
a = ["Python", "is", "programming", "language"]
s = a[1:5:3]
print(s)
```

#### 리스트(List)
* 여러 개의 데이터를 하나의 변수에 순서대로 담을 수 있는 자료형입니다
* 대괄호 `[]`를 사용하여 생성하며, 각 요소는 쉼표 `,`로 구분합니다.

</br>

#### 인덱싱(Indexing)
* 리스트 안에 있는 요소들의 위치 번호(인덱스)를 의미합니다.
* 파이썬의 인덱스는 0부터 시작(Zero-based)한다는 점이 가장 중요합니다.

|인덱스(Index) | 0 | 1 | 2 | 3 |
|-----------|---|---|---|---|
|데이터 (값) | "python" | "is" | "programming" | "language" |

</br>

#### 슬라이싱(Slicing)
* 리스트의 일부분을 잘라내어 새로운 리스트를 만드는 기능입니다.
* 기본문법: `리스트[시작_인덱스 : 끝_인덱스 : 간격(Step)]`
    * 시작_인덱스: 추출을 시작할 위치입니다. (이 위치의 값은 포함됩니다.)
    * 끝_인덱스: 추출을 종료할 위치입니다.(이 위치의 값은 포함되지 않으며,그 바로 앞까지만 추출 합니다.)
    * 간격(Step): 몇 칸씩 건너뛰며 추출할지 결정합니다.

중요한 특징: 파이썬의 슬라이싱은 지정한 '끝_인덱스'가 리스트의 실제 길이를 벗어나더라도, 에러(IndexError)를 발생시키지 않고 리스트의 끝까지만 안전하게 처리합니다.


### 1. 리스트 생성 및 할당
```python
a = ["Python", "is", "programming", "language"]
```
* 4개의 문자열 데이터를 가진 리스트가 메모리에 생성됩니다.

* 이 리스트는 변수 a에 저장됩니다.

* 이때 각 데이터의 인덱스는 다음과 같습니다: a[0]="Python", a[1]="is", a[2]="programming", a[3]="language".



### 2. 리스트 슬라이싱 연산
```python
s = a[1:5:3]
```
1. **시작 인덱스(1)**: 인덱스 1번인 `"is"`부터 추출을 시작합니다.
2. **간격(3)**: 그 다음 요소를 찾기 위해 인덱스에 3을 더합니다. (1 + 3 = 4), 즉, 다음으로 추출할 대상은 인덱스 4번 입니다.
3. **끝 인덱스 검사(5)**: 파이썬은 인덱스 4번 값을 가져오려고 시도합니다.  
끝 인덱스가 5이므로 조건(5보다 작을 것)에 맞습니다.
4. **리스트 길이 초과**: 하지만 리스트 `a`의 마지막 인덱스는 3입니다. 인덱스 4번에는 데이터가 존재하지 않습니다.
5. **추출 종료**: 파이썬 슬라이싱은 데이터가 없는 범위를 만나면 에러를 내지 않고 거기서 추출을 멈춥니다.
6. **결과저장**: 최종적으로 추출된 값이 `"is"`하나만 담긴 새로운 리스트 `["is"]`가 만들어지고, 변수 `s`에 저장됩니다.

</br>

### 3.결과 출력
```python
print(s)
```
* 변수`s`에 저장된 리스트를 화면에 출력합니다.
* 최종 출력 결과 `['is']`
