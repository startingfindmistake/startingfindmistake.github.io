## OSI 7계층

---

#### 라우팅

* 정적라우팅 => 수동 경로 설정
* RIP => 홉수 기반 거리 백터 방식
* OSPF => 상태 기반 링크 방식
* BGP => AS간 라우팅 사용

**1. 물리 계층**

* 데이터 단위: <span style="color: #FF69B4; font-weight: bold;">비트(Bit) </span>

* 전기 신호, 케이블, 커넥터
* RS-232C

</br>
</br>

**2. 데이터 링크 계층**

* 데이터 단위: <span style="color: #FF69B4; font-weight: bold;">프레임(Frame) </span>
* Mac주소
* HDLC,Ethernet,ppp

</br>
</br>

**3. 네트워크 계층**

* 데이터 단위: <span style="color: #FF69B4; font-weight: bold;">패킷(Packet) </span>
* **IP주소**
* ICMP, IGMP, ARP, RARP, DNS, DHCP
* **라우팅**
* 논리 주소 사용

</br>
</br>

**4. 전송 계층**

* 데이터 단위: <span style="color: #FF69B4; font-weight: bold;">세그먼트(Segment) </span>
* **TCP, UDP**
* 종단 간 신뢰성
* 흐름제어

</br>
</br>

**5. 세션 계층**

* 데이터 단위: Data
* 대화관리, 동기점

</br>
</br>

**6. 표현 계층**

* 데이터 단위: Data
* 암호화, 압축 코드 변화

</br>
</br>

**7. 응용 계층**

* 사용자 인터페이스
* HTTP, HTTPS
* DNS
* SMTP, SNMP
* FTP
* TELNET
