이 문제를 보면은 첫 번째 접근에서 볼수 있듯이 값을 찾아야 하는데 2가지 방법이 존재할수 있다.

그중 우리는 `charAt()`메서드를 사용하여 문제를 풀어볼 예정이다.

```java
import java.util.Scanner;

public class test {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

        // charAt()메서드 자체가 하나의 긴 'string'안에서서 특정 위치(인덱스)의 글자를 하나씩 빼오기 때문에 배열이 필요 없음
        String text = sc.next();
        int i = 0;

        // 반복문을 통해 하나씩 비교를 해보자
        for(i = 0; i < text.length(); i++) {
            if(text.charAt(i) == 'c') {
                if(text.charAt(i+1) == '=' || text.charAt(i+1) == '-'){
                    // 맞다면 다음 글자로 숫자 카운트를 하고 그것을 건너 띄자
                    i += 2;
                    // 이렇게 하면 if문을 나오면 어디로 가지? 갑자기 헷갈리네
                } else {
                    System.out.println("2번째 안쪽 if문");
                }
            } else {
                System.out.println(i);
                System.out.println("1번째 if문");
            }
        } System.out.println("for문 밖");
        System.out.println(i);
    }
}


```
